# pluto
# Плуто

Терминальный AI-компаньон с характером. Groq API.

## Установка

```
pip install -r requirements.txt
```

## Запуск

```
nano .env
./pluto.py
```

## Режимы

- `/mode daily` — повседневное общение
- `/mode coding` — фокус на код
- `/mode kind` — добрый и тёплый
- `/mode linux` — линуксоид-фанат FOSS

## Команды

`/search`, `/fetch`, `/read`, `/write`, `/edit`, `/ls`, `/clear`, `/theme`
